# 自动化脚本使用说明

## 【元数据】
- **用途**：说明如何使用自动化脚本提升工作效率
- **创建日期**：2026-01-03
- **版本**：V1.0

---

## 📚 脚本列表

### 1. 自动更新工作上下文.py
**用途**：自动更新工作上下文文件，记录任务完成情况

**用法**：
```bash
python 自动更新工作上下文.py <任务描述> [状态] [进度]
```

**示例**：
```bash
python 自动更新工作上下文.py "创建优化工具文件" "已完成" 100
```

---

### 2. 自动更新任务清单.py
**用途**：自动更新任务清单文件，跟踪任务状态和进度

**用法**：
```bash
# 添加任务
python 自动更新任务清单.py add <TASK-ID> <描述> <优先级>

# 更新任务状态
python 自动更新任务清单.py update <TASK-ID> <状态> [进度]
```

**示例**：
```bash
python 自动更新任务清单.py add TASK-010 "测试自动化脚本" P1
python 自动更新任务清单.py update TASK-010 "已完成" 100
```

---

### 3. 智能缓存管理器.py
**用途**：实现基于文件变更、依赖关系、优先级的智能缓存机制

**用法**：
```bash
# 检查缓存
python 智能缓存管理器.py check <文件路径> [优先级]

# 更新缓存
python 智能缓存管理器.py update <文件路径> <检查类型> <结果> [优先级]

# 更新Markdown文件
python 智能缓存管理器.py update-md
```

**示例**：
```bash
python 智能缓存管理器.py check "RAG知识库/README.md" P1
python 智能缓存管理器.py update "RAG知识库/README.md" "全面检查" "通过" P1
```

---

### 4. 自动化工作流集成.py
**用途**：集成所有自动化脚本到工作流程中

**用法**：
```bash
python 自动化工作流集成.py complete <TASK-ID> <任务描述> [文件路径] [检查类型] [结果] [优先级]
```

**示例**：
```bash
python 自动化工作流集成.py complete TASK-010 "创建优化工具文件"
python 自动化工作流集成.py complete TASK-011 "检查RAG知识库" "RAG知识库/README.md" "全面检查" "通过" P1
```

## 🔄 工作流程集成

### 标准工作流程

```
1. 执行任务
   ↓
2. 任务完成
   ↓
3. 调用自动化工作流集成.py
   ↓
4. 自动更新工作上下文
   ↓
5. 自动更新任务清单
   ↓
6. 自动更新检查缓存（如果有）
   ↓
7. 完成
```
