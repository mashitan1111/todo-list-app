# 工作待办清单应用 - 启动说明

## ✅ 推荐启动方式

### 方式 1：双击批处理文件（推荐）
1. 找到文件：`启动工作待办清单.bat`
2. **双击运行**
3. 会打开一个新窗口（标题为 "Todo App"）
4. 等待几秒，浏览器会自动打开 `http://127.0.0.1:5000`
5. **保持命令窗口打开**，关闭窗口会停止应用

### 方式 2：命令行运行
如果双击不行，可以：
1. 按 `Win + R`，输入 `cmd`，回车
2. 运行：
```cmd
cd /d "C:\Users\温柔的男子啊\Desktop\crusor\圆心工作\工具和脚本\工具脚本"
python 工作待办清单桌面应用_精美版.py
```
3. 在浏览器访问：`http://127.0.0.1:5000`

## ⚠️ 常见问题

### 问题 1：双击后窗口秒退
**解决方案：**
- 右键点击 `启动工作待办清单.bat`
- 选择"以管理员身份运行"

### 问题 2：显示"无法解析导入 flask"
**说明：** 这是 Cursor IDE 的警告，不影响运行。Flask 已经安装，应用可以正常运行。

### 问题 3：端口 5000 被占用
**解决方案：**
1. 关闭占用 5000 端口的程序
2. 或者修改应用代码中的端口号

### 问题 4：应用启动后立即停止
**可能原因：**
- 端口被占用
- 文件路径问题
- Python 环境问题

**解决方案：**
- 查看命令窗口中的错误信息
- 确保所有依赖已安装：`pip install flask`

## 📝 使用提示

1. **保持窗口打开**：应用运行时，命令窗口必须保持打开
2. **停止应用**：按 `Ctrl+C` 或关闭命令窗口
3. **访问应用**：浏览器会自动打开，或手动访问 `http://127.0.0.1:5000`

## 🔧 技术说明

- **Python 版本**：3.14.2
- **Flask 版本**：3.1.2
- **端口**：5000
- **数据库**：SQLite (tasks.db)
